# Login to GitHub CLI

2Ô∏è‚É£ Repository Management
# Clone a repo
gh repo clone username/repo

# Create a new repo (interactive)
gh repo create

# Create a new repo public/private non-interactively
gh repo create myrepo --public --description "My DevOps repo"

# Fork a repo
gh repo fork owner/repo --clone=true

# List repos of a user/org
gh repo list username --limit 50

# View repo details in browser
gh repo view username/repo --web

3Ô∏è‚É£ Branch & PR Management
# List pull requests
gh pr list --state open --assignee @me

# View a PR
gh pr view 123
gh pr view 123 --web  # opens in browser

# Checkout PR branch locally
gh pr checkout 123

# Create a new PR from current branch
gh pr create --title "Add feature X" --body "Detailed explanation" --base main

# Merge a PR
gh pr merge 123 --merge       # Merge commit
gh pr merge 123 --squash      # Squash commits
gh pr merge 123 --rebase      # Rebase merge

# Close a PR without merging
gh pr close 123 --comment "Closing this PR because it's obsolete"

4Ô∏è‚É£ Issue Management
# List issues assigned to you
gh issue list --assignee @me --state open

# Create a new issue
gh issue create --title "Bug in login" --body "Steps to reproduce:\n1. ...\n2. ..." --label bug

# View issue details
gh issue view 42

# Close an issue
gh issue close 42 --comment "Fixed in commit abc123"

5Ô∏è‚É£ Workflow / Actions (CI/CD)
# List workflows in a repo
gh workflow list

# Run a workflow manually
gh workflow run ci.yml --ref main

# View recent workflow runs
gh run list --limit 10

# View logs of a specific run
gh run view 123 --log

# Watch a workflow run in real time
gh run watch 123

# Cancel a running workflow
gh run cancel 123


Pro Tip: Combine gh run list with --json and jq to filter and automate monitoring.

6Ô∏è‚É£ Secrets Management
# List repository secrets
gh secret list

# Add a new secret
gh secret set MY_SECRET --body "my-secret-value"

# Remove a secret
gh secret remove MY_SECRET


Useful for CI/CD pipelines (Docker passwords, API tokens, SSH keys).

7Ô∏è‚É£ Gists (Useful for sharing scripts / configs)
# Create a public gist from file
gh gist create script.sh --public

# Create a secret gist
gh gist create script.sh --secret

# List your gists
gh gist list

# View gist in browser
gh gist view <gist-id> --web

8Ô∏è‚É£ Aliases (DevOps speed hacks)
# Set alias for checkout PR
gh alias set co 'pr checkout'

# Open a repo in browser
gh alias set web 'repo view --web'

# Quickly merge PRs
gh alias set pm 'pr merge --squash'

# List workflow runs with JSON output
gh alias set runs 'run list --limit 5 --json databaseId,status,conclusion'
